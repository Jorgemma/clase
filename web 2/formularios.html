Introduccion

Es un control creado con el tag FROM , el cual permite obtener informacion de entrada desde el usuario


artributos importantes
  • ACTION: establece la URL destino de los datos
  • METHOD: establece el verbo del protocolo HTTP con el cual se enviara la imformacion. los mas usados son GET y POST
  • NAME: asigna un nombre al formulario paara ser recuperado del lado del servidor
