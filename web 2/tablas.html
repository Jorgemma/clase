Tablas

Con los Tag's TABLE, TR, TH, y TD se crean tablas que organizan Informaccion para resumirla

    TR: permite declarar renglones TH: permite declarar cabeceras de columna TD: permite declarar los datos de las celdas

Los artributos: border, caption, colspan y rowspan

  • border: establece el ancho para la tabla
  • caption: establece un titulo arriba de la cabecera
  • colspan: permite que una celda ocupe varias columnas
  • rowspan perimite que una celda ocupe varios renglones
